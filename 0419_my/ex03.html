display : inline
  • 한 줄 안에 여러 요소를 구분하여 배치할 때 사용한다
  • 대표적으로 span이 있다
  • 크기를 직접 지정할 수 없다
display : block
  • 줄 단위로 구분되는 영역을 생성할 때 사용한다
  • block 요소는 혼자서 한 줄을 차지한다(h1, div, p ...)
  • 너비와 높이를 직접 지정할 수 있다

display : inline-block
  • inline 처럼 한 줄에 여러 요소를 배치할 수 있다
  • block 처럼 너비와 높이를 직접 지정할 수 있다
  • 크기를 지정하여 여러 요소를 한 줄에 배치하고 싶을 때 사용한다
